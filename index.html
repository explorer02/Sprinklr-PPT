Main Project

Sprinklr Blogs

https://blog-explorer.netlify.app/

What does it do??

A user can write a blog in either Markdown Format or an Online Editor. The blog will then be published online. This website classifies blogs into different categories (Webpack, Machine Learning, Most Recent, Most Popular, etc.)

All Features

  • Clean and Simple UI
  • Lightweight website
  • Quick Navigation within Website
  • Search within Blogs
  • Filtering Blogs using Analytics
  • Syntax Highlighting for Code Snippets
